# Контекст проекта для Claude Code

## Что это
Переписанный сайт **ФБУЗ «Центр гигиены и эпидемиологии в Мурманской области»** (fguzmo.ru).
Старый сайт (CMS Parser3) — устаревший, заменяем на современный статический сайт.
Размещение: GitHub Pages (репозиторий уже создан и захостен). Хостинг на будущее — nic.ru / vk.cloud.
Владелец — не программист, обновлять должен сам.

## Стек и архитектура (важно соблюдать)
- Чистый статический сайт: HTML + CSS + ванильный JS. Без сборщиков и фреймворков.
- Все пути относительные (работает и локально через file://, и на GitHub Pages в подпапке).
- **Главный принцип: весь редактируемый контент — в одном файле `assets/content.js`** (объект `window.SITE`).
  Дизайн и логика — в `assets/styles.css` и `assets/app.js`, их владелец не трогает.
- `app.js` сам строит шапку/меню/подвал на каждой странице и рендерит данные из `content.js`.
- Палитра: синий `#0d3b7a` / `#1a5cb5`, жёлтый `#f5c400`. Шрифты Montserrat + PT Sans.
- Логотип — герб Роспотребнадзора, `assets/logo.jpg`.

## Структура файлов
- `index.html` — главная (герой, быстрые действия, услуги, новости, документы, контакты, горячие линии)
- `uslugi.html` — услуги и цены (прайс из `content.js`, FAQ)
- `dokumenty.html` — документы (список из `content.js`, файлы в папке `files/`)
- `novosti.html` — новости (сейчас из `content.js`; задел под авто-ленту ВК)
- `kontakty.html` — контакты + карта (Яндекс) + горячие линии
- `ob-organizacii.html` — об организации: о центре, руководство, ФИЛИАЛЫ (выше направлений), направления, отделы, сведения/раскрытие информации
- `antikorrupciya.html` — противодействие коррупции (обязательный раздел)
- `rukovoditel.html?id=...` — карточка руководителя с биографией (шаблон, данные из `content.js` → `leadership`)
- `filial.html?id=...` — карточка филиала: график приёма, часы, адрес, телефоны, структура подразделений, положение (данные из `content.js` → `branches`)
- `assets/` — `content.js`, `app.js`, `styles.css`, `logo.jpg`, фото руководителей (`ruk-*.jpg`)
- `files/` — реальные документы (прейскурант-2026, ЛМК, политика ПД, лицензия, устав, ИНН, ОГРН, положения о филиалах и др.)
- `.nojekyll` — чтобы GitHub Pages отдавал всё как есть

## Особенности реализации
- Версия для слабовидящих — встроена (кнопка в верхней панели), классы на `<html>` (bvi, схемы, размер шрифта), сохранение в localStorage.
- Телефоны: функция `phoneLinks()` в `app.js` разбивает строку с несколькими номерами на отдельные кликабельные ссылки (каждый с новой строки), чистит «тел./факс», дедуплицирует.
- Структура филиала: группировка по отделам, плотная раскладка колонками (без растягивания коротких карточек). Все номера сохранены, в т.ч. по разным городам.
- Карточки руководителей/филиалов кликабельны, ведут на `rukovoditel.html`/`filial.html` с `?id=`.
- Заглушка-аватар (SVG) подставляется, где нет фото.
